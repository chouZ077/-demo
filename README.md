# 高校周边餐饮地图 Demo

一个可直接打开运行的静态 Web App Demo，用地图方式查看高校周边餐饮店。

## 怎么打开

直接打开 `index.html` 即可，不需要安装任何依赖。

## 怎么使用

1. 在主页点击高校名称，进入该学校周边餐饮视图
2. 继续缩放或点击店铺，查看店铺详情
3. 左侧可以按菜系打开分类列表
4. 顶部可以搜索店名
5. 详情页里可以看评分、评论摘要和“大学生严选”投票

## 文件说明

- `index.html`：页面结构
- `styles.css`：页面样式和动画
- `app.js`：交互逻辑
- `docs/`：需求、设计和交互说明文档
